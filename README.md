# Namma-Metro-Sahaya-Infrastructure-

## Overview

Namma Metro Sahaya is a smart metro navigation and assistance application developed to simplify urban metro travel for commuters. The application helps users identify the shortest and most efficient metro routes between stations using graph-based algorithms such as BFS and Dijkstra’s Algorithm. It provides route guidance, interchange details, estimated travel time, fare calculation, and nearby station information to make commuting easier and more organized.

The project is designed for daily commuters, students, tourists, and first-time metro users who need quick and accurate metro navigation support.

## System Architecture 📌

The application represents metro stations as nodes and metro routes as edges in a graph data structure.
Using BFS and Dijkstra’s Algorithm, the system calculates:

* Shortest route
* Minimum travel distance
* Efficient interchange path

---

## How It Works ⚙️

1. User selects source station.
2. User selects destination station.
3. The application processes metro routes using graph algorithms.
4. The shortest and most efficient path is displayed.
5. User receives:

   * Route details
   * Interchange stations
   * Estimated fare
   * Estimated travel time
